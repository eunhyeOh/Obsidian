## 예약 가능 날짜/좌석 API

- Endpoint
        - 날짜에 따라 예약 가능한 좌석 목록 조회
        - URL: `/api/v1/concerts/{concertId}/available-seats`
        - Method: GET
- Request
    - Headers : Authorization: Bearer
    - Params : concertId(콘서트 ID)
    - Query(좌석 조회) : date
- Response
    - HTTP Status Codes
        
        2. 200 OK : 예약 가능 좌석 조회
        
        ```
          {
             "concertId": 67890,
             "selectedDate": "2025-01-10",
             "availableSeats": [
             ["" : 1, "": "A1"],1, 2, 3, 4, 5]
           }
        ```
        
        - 404 Not Found
        
        ```
          {
            "error": "콘서트 정보를 찾을 수 없습니다."
          }
        ```
        
        - 401 Unauthorized
        
        ```
          {
            "error": "잘못된 토큰 정보입니다."
          }
        ```
        
- Authorization
    - 필수 : 토큰 인증 필요
